Projeto desenvolvido com o objetivo de praticar Java e Spring Boot, aplicando na prática conceitos como API REST, separação em camadas (Controller, Service, Repository), JPA e injeção de dependência.

Foco em consolidar fundamentos e evoluir gradualmente no desenvolvimento backend.

## Estrutura de Pastas

Arquitetura em camadas para melhor organização e manutenção:

- `service` — regras de negócio  
- `controller` — camada de controle  
- `entity` — entidades JPA  
- `repository` — acesso a dados com Spring Data
